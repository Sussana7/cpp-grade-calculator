# C++ Grade Calculator

This is a simple console-based C++ program that allows a user to input their scores for five academic subjects and then outputs the corresponding letter grades based on a grading scale.

| 60 - 64     | C     |
| 55 - 59     | D+    |
| 50 - 54     | D     |
